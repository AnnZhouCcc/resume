Assessment Data Analytics Site

A responsive data analytics site with visually appealing graphs designed with D3.js, facilitating over 1000 clients to gain more robust insights into student population by applying any chosen filter combinations (e.g. schools, races, etc.).

An internship project done with The Whether

Formulated interactivity between filter selection and data fetching with Vue.js, thus all graphs reflected any change in the filter combinations instantly and spontaneously.

Optimized website performance, minimizing the required times of data fetching to 12.5% of the original algorithm, by incorporating a data fetching component and restructuring chart components.

Enabled default selection and generation of deep-linked page with Vue Router, thus users were able to share with others graphs of their chosen filter combinations directly by sharing the url (with embedded query parameters); carefully handled a variety of complicated or ambiguous cases of selection.

  • Date: June 2018
  • Languages & Frameworks: Vue.js, D3.js, JavaScript, HTML5, CSS
